# Writing Templates

Ten templates for daily book-based writing. Each template has **3 tiers** — the tutor
picks the tier from the student's profile. Higher tiers earn bonus XP (see
`game/rules.md`).

## Picking by today's reading amount

| Read today | Best fits |
|------------|-----------|
| Just a few pages | 09 Quick Write 3-2-1 · 07 Golden Line · 02 Prediction |
| About a chapter | 01 Summary · 03 Character Diary · 04 Opinion · 05 Letter · 08 Connection |
| A lot / a big event happened | 06 Rewrite the Scene · 04 Opinion · 01 Summary |
| Finished the book! | 10 Book Review |

## Index

| # | Template | Skill it builds |
|---|----------|----------------|
| 01 | Summary | Main idea, sequencing |
| 02 | Prediction | Inference, evidence |
| 03 | Character Diary | Point of view, voice |
| 04 | Opinion / Response | Claim + evidence + explanation |
| 05 | Letter to a Character | Audience, empathy |
| 06 | Rewrite the Scene (What If…) | Creativity, plot logic |
| 07 | Golden Line | Close reading, quoting |
| 08 | Connection (Text-to-Me/World) | Analysis, personal voice |
| 09 | Quick Write 3-2-1 | Habit keeper for short days |
| 10 | Book Review | Evaluation, persuasion |
