# 03 — Character Diary

**Skill:** point of view, voice, empathy
**Best when:** something big happened to a character today

Write a diary entry **as the character**, about what happened in today's reading.
Use "I". Start with *Dear Diary,* if you like.

## Tier 1 (4–5 sentences)

- Today, ______ happened to me.
- I felt ______ because ______.
- Tomorrow I hope ______.

✅ Checklist: stays in "I" the whole time, names one real event from the book.

## Tier 2 (1 paragraph)

- Describe the event **from inside the character's head** — what they saw, heard, felt
  in their body (heart pounding? hands shaking?).
- Include one **worry** and one **hope**.
- Mention another character and how the diary-writer feels about them right now.

✅ Checklist: at least 2 feeling moments shown (not just "I was sad"), another
character mentioned.

## Tier 3 (1–2 paragraphs)

- Match the **character's voice**: would they use slang? Short angry sentences?
  Fancy words? Write like THEY would write.
- Include something the character would **never say out loud** — a secret thought.
- Reference a small detail from the book that proves you read closely.

✅ Checklist: voice sounds like the character (not like you), one secret thought,
one precise book detail.
